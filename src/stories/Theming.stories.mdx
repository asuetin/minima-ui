# Theming

All components in the library support theming through the styled-components theme object.

The part of the theme object used by Minima has the following signature:

```tsx
export type ColorType = [CSSProperties['color'], ...CSSProperties['color'][]] & {length: 10};
export type SizeType = [CSSProperties['width'], ...CSSProperties['width'][]] & {length: 5};
export type ShadowType = [CSSProperties['boxShadow'], ...CSSProperties['boxShadow'][]] & {length: 3};

export type ThemeType = {
	fontFamily: CSSProperties['fontFamily'];
	background: ColorType;
	accent: ColorType;
	content: ColorType;
	size: SizeType;
	radius: SizeType;
	shadow: ShadowType;
	light: ShadowType;
	focus: CSSProperties['boxShadow'];
};
```
